## SEO Setup

All SEO metadata lives in `src/layouts/Layout.astro`:
- Title, description, canonical, Open Graph, Twitter Card
- Two JSON-LD blocks: `SoftwareApplication` and `Organization`
- `sitemap.xml` auto-generated by `@astrojs/sitemap` on build (requires `site` set in `astro.config.mjs`)

## Design System

- **Font:** Plus Jakarta Sans (Google Fonts, loaded in `global.css`)
- **Primary color:** `brand-600` = `#16a34a` (green)
- **Accent color:** `egg-500` = `#f59e0b` (amber/egg yellow)
- Custom color scales `brand` and `egg` defined in `tailwind.config.mjs`
- All text in Latin American Spanish (`lang="es-419"`)

## Key Constraints

- No JS frameworks ‚Äî keep it pure Astro (zero hydration)
- All copy targets non-technical farmers; keep language simple
- `site` in `astro.config.mjs` must match the production domain for sitemap and canonical URLs
- Update the WhatsApp link in `Pricing.astro` and `Footer.astro` with the real phone number before going live
